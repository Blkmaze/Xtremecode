## ğŸ” Troubleshooting

### Installation Issues
- **"Install Failed"**: Enable Unknown Sources in Settings
- **"Parse Error": Download APK again (corrupted file)
- **"App Not Installed": Clear storage and retry

### Streaming Issues
- **Buffering**: Check internet speed (5+ Mbps recommended)
- **No Channels**: Verify server URL and credentials
- **Audio Issues**: Check device volume and try different channels

### Performance Tips
- **Close other apps** to free up memory
- **Restart Fire Stick** if app is slow
- **Check internet connection** speed
